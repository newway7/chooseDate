## 自定义时间区间以供用户选择

### 个人使用jquery写的一个小插件，只需要给函数一个开始时间和结束时间即可。


#### 用法：
```
        $('#startDate').kinerDatePicker({
            startDate: '2019-02-03',//开始时间
            endDate: '2019-09-20',//结束时间
            clickMaskHide: true,//点击半透明区域隐藏时间选择器
        });
```
